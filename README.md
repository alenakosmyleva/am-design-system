# am-design-system
Design system monorepo for multi‑product enterprise platform. Includes tokens, Angular components, patterns, and Storybook documentation.

## Goals
- Provide consistent UI across multiple internal products
- Centralize design tokens, components, and patterns
- Enable scalable contribution workflow for designers and developers

## Tech Stack
- Angular
- Storybook
- Design tokens (JSON)
- SCSS + CSS variables
- Monorepo structure

## Status
🚧 Initial setup phase. Foundations in progress.
